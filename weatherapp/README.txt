System Requirements:

Node JS
Visual Studio Code

Steps to run the application:

1.Open terminal.Navigate to the project folder in the terminal. 
Run the command:  npm install
2.Once the installation of libraries for this folder is done run the following command:
npm install -g concurrently
3.Enter the command: cd weatherapp
4.Enter the command: npm install
5.Enter the command: cd..
6.Enter the command: npm run dev
7.Navigate to localhost:3000 in the browser to see the app in action
